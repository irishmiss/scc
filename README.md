
## Syntactic Code Converter

This plugin may help you convert HAML, SASS and LESS to required directory.

### Requirements

* Mac OS X or Linux
* Installed ruby
* Installed rubygems

### Usage:

_config.yml - pre-configuration
Configure primary assets config - _compile.yml

### Compile scripts:

```html
rake compile
```
